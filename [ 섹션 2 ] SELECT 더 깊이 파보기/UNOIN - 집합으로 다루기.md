# 🍀 UNION - 집합으로 다루기

| 연산자      | 설명                      |
| ----------- | ------------------------- |
| `UNION`     | 중복을 제거한 집합        |
| `UNION ALL` | 중복을 제거하지 않은 집합 |

## 🧸 합집합

- WHERE 조건에 해당하는 값 모두 가져오기

## 🧸 교집합

- AND 조건에 해당하는 두 조건에 모두 해당하는 값만 가져오기

## 🧸 차집합

- A와 B중 A와 B의 공통, B를 제외하고 A의 값만 가져오기

## 🧸 대칭차집합

- A와 B중 A와 B의 공통 값만 제외하고 가져오기
